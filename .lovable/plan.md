

# Página Simples com Texto

## O que será criado:

### Página Principal
- **Fundo branco** limpo e minimalista
- **Seção de texto** centralizada com a palavra "oi"
- **Texto em preto** para máximo contraste e legibilidade

### Design
- Layout simples e limpo
- Texto centralizado na página
- Estilo minimalista

Esta é uma página básica que pode servir como ponto de partida para expandir com mais conteúdo depois.
